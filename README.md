# 💰 Expense Tracker App with ML (Streamlit)

An intelligent expense tracking web application built using **Streamlit** and **Machine Learning**. This app helps users log daily expenses, categorize them automatically, and gain insights into spending patterns.

---

## 🚀 Features

- 📊 **Interactive Dashboard** with Streamlit
- 🧾 **Expense Logging** (amount, category, date, description)
- 🤖 **ML-based Auto Categorization** of expenses
- 📈 **Spending Insights & Visualizations**
- 🔍 **Search & Filter Expenses**
- 💡 **Smart Suggestions** based on past spending
- 💾 **Data Persistence** (CSV / Database support)

---

## 🧠 Machine Learning Component

The app uses a trained ML model to automatically classify expenses into categories like:

- Food 🍔
- Travel 🚗
- Shopping 🛍️
- Bills 💡
- Entertainment 🎬
- Others

### Model Details:
- Algorithm: (e.g., Logistic Regression / Naive Bayes / Random Forest)
- Text preprocessing: Tokenization, Stopword removal, TF-IDF
- Input: Expense description
- Output: Predicted category

---

## 🛠️ Tech Stack

- **Frontend/UI:** Streamlit
- **Backend:** Python
- **ML Libraries:** Scikit-learn, Pandas, NumPy
- **Visualization:** Matplotlib / Seaborn / Plotly
